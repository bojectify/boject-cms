# Performance

This directory mirrors the operator-facing sections of the load-test reports
generated by [`@boject/perf`](../../perf/README.md). Use it for a quick read of
what the system can sustain and what we recommend operators configure;
follow the links for the underlying data.

- Full harness: [`perf/README.md`](../../perf/README.md)
- Full reports (summary + CSV + plots): [`perf/reports/`](../../perf/reports/)
- Latest operator summary:
  [`2026-04-28-report-summary.md`](./2026-04-28-report-summary.md)

When a new sweep lands, copy the **Headline numbers**, **Recommendations**,
and **Run notes** sections of `perf/reports/<run-id>/summary.md` into a new
file in this directory and update the link above.
